Easy-Peasy Git Guide

Learn how to connect your computer, configure Git, and upload your files step-by-step!

What is Git? Git is like a magical time-machine folder on your computer. It keeps track of every single change you make to your files, so you can never lose your work!

1. Username & Password (Authentication)

When your computer communicates with this server (to upload or download files), it will pop up a window asking for your credentials:

  • Username: Use the exact username you use to log in to this website (e.g., {{ request.user.username }}).
  • Password: Use the exact password you use to log in to this website.

2. One-Time Setup (Introduce Yourself to Git)

Before you do anything, Git needs to know who you are. Open your terminal or command prompt and copy-paste these commands: 

git config --global user.name "{{ request.user.display_name|default:request.user.username }}"
git config --global user.email "{{ request.user.email|default:'user@example.com' }}"
git config --global safe.directory "*"
* Note: This prevents "dubious ownership" errors when accessing folders owned by other server users.

3. Step-by-Step Example (How to save and send files)

Step A: Download a Repository to your PC

Click any repository card, copy its "Clone URL", and run this command in terminal:

git clone [CLONE_URL_HERE]
Step B: Make Changes & Save Locally

Place your project files inside that new folder, then save the changes in Git:

git add .
git commit -m "Added my project files"
Step C: Send to the Web Server

Push your files so others can see them on this dashboard:

git push origin master
