Git Installation & System Path Guide

Who needs Git installed? (Server vs. Clients)

Since this application is deployed on a centralized server and accessed by multiple client PCs via the server's IP address:

Server PC REQUIRED: The computer/PC running the web server application hosting IIAP OM must have Git installed in its system PATH. Without it, repository operations will fail system-wide.
Web Users NOT REQUIRED: Standard users visiting the site via their browser on client PCs (Windows, macOS, Linux) to browse files or drag-and-drop uploads do not need Git on their local machines.
CLI Developers REQUIRED (Locally): Developers who wish to clone, push, or pull using local terminal commands (e.g. git clone http://IP:PORT/) do need Git installed locally on their own PC.

Select the operating system of the target machine (the Server PC or local developer machine) to get installation instructions:

On Ubuntu or Debian-based Linux servers, install Git using the package manager: 

sudo apt update
sudo apt install git -y

Verify the installation and path using: 

git --version
which git
Tip: If installing via script or non-standard paths, make sure the directory containing the git executable is appended to the system PATH variable in /etc/environment or ~/.bashrc.
1. Download & Install Git: Download the official Git for Windows installer from git-scm.com and run it.
2. Choose the correct PATH option: During installation, on the "Adjusting your PATH environment" screen, make sure to select:
"Git from the command line and also from 3rd-party software"
3. Adding to Environment Variables Manually: If Git commands are still not found, add it to your System PATH:
  1. Open the Start Menu, search for "environment variables", and choose "Edit the system environment variables".
  2. Click "Environment Variables...".
  3. Under "System variables", find "Path" and click "Edit...".
  4. Click "New" and paste the path to your Git cmd folder, which is typically:
    C:\Program Files\Git\cmd
  5. Click "OK" on all windows to save the settings.
Crucial Step: Once PATH is updated, you must restart your terminal, command prompt, or development server (e.g. running python manage.py runserver) for changes to take effect.

On macOS, you can install Git using Homebrew or Apple's developer command tools:

Option 1: Using Homebrew (Recommended) 
brew install git
Option 2: Using Xcode Tools

Simply run this command to trigger macOS command line tools downloader: 

xcode-select --install
